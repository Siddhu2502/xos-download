# xos-download


## Steps to download working thing of XOS

In case you are not able to run xos, use these steps



### Step 1
open terminal and go to a desired location

```
cd Downloads
```

### Step 2

Download xos

```
wget ....
```

### Step 3

unzip it

```
tar -xvf xoz.gz
```

### Step 4

Move the file to home directory so that its uniform for all

```
mv myxos-1/ ~/myxos
```

Done :)
